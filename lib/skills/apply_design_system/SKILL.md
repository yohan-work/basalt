# Apply Design System

Injects appropriate classes or styles to match the project's design guidelines.

## Inputs
-   `componentPath`: Path to the component file.

## Outputs
-   Success message.

## Instructions
1.  Analyze the component's current structure.
2.  Add utility classes (Tailwind) or CSS modules that align with the Global Design System (colors, spacing, typography).
3.  Do not break existing functionality.
