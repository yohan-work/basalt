# Search NPM Package

Searches for NPM packages that match the user's need.

## Inputs
-   `query`: Keywords or description of the functionality needed.

## Outputs
-   Name of the recommended package (e.g., 'lucide-react').

## Instructions
1.  Identify the best standard package for the `query`.
2.  Prioritize popular, well-maintained packages.
3.  Return the package name and version if possible.
