# Refactor Code

Takes existing code and applies best practices to improve it.

## Inputs
-   `code`: The original code snippet or file content.
-   `focus`: (Optional) What to focus on (e.g., 'readability', 'performance', 'types').

## Outputs
-   The refactored code string.

## Instructions
1.  Analyze the code for code smells or inefficiencies.
2.  Apply changes according to the `focus` area.
3.  Ensure logical correctness is maintained.
